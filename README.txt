Media Preview for Alfresco Share
================================

Author: Will Abson

This add-on project for Alfresco Share provides the ability to preview audio 
and video files within the Share user interface.

Specifically the project provides enhancements to web preview component to 
support Flash-based previews of audio and video content. Also bundles 
thumbnail registry changes required to support H264/FLV thumbnails for video 
content and MP3 thumbnails for audio content.
